Eclipse 4

Instead of using the Extension Point mechanism, EMF Components leverages from DSL and Google Guice Injection.

Because of this, it is very easy to use it with Eclipse 4.

Just be sure to start (as illustrated in section Getting Started) with a "Eclipse Modeling Tools" package from Eclipse Downloads

of course you will have to install the E4 Tools, e.g. from the following update site:

http://download.vogella.com/juno/e4tools
